# EisiEsn360FrtAngular

## Build

- Standard build (cross-platform): `npm run build`
- If you are forced to use Node 17+ with OpenSSL legacy mode: `npm run build:legacy-openssl`

Node version expected by this project:
- `.nvmrc`: `14.21.3`
- `.node-version`: `14.21.3`

Deploy to docs directory:
- `ng build --output-path docs --base-href .`
