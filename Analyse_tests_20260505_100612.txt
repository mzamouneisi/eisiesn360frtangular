CATEGORIE B - No provider for ActivatedRoute (cause: RouterTestingModule absent)
  Impact : ~15 specs
  Specs concernees :
    ActivityFormComponent, ActivityTypeFormComponent, AddMultiDateComponent,
    CategoryFormComponent, CraConfigurationComponent, DocCategoryFormComponent,
    EsnArboComponent, MsgFormComponent, MsgHistoFormComponent,

  Correctif :

    fournir { provide: ActivatedRoute, useValue: { snapshot: {}, params: of({}) } }

CATEGORIE C - No provider for MatDialog (cause: UtilsIhmService injecte MatDialog)
  Impact : ~10 specs (toutes celles qui injectent DataSharingService ou UtilsIhmService)
  Specs concernees :

    SignupDialogComponent, UploadFileComponent, UserConnectedComponent,
    ConsultantArboComponent, MsgNotificationsComponent
  Erreur type :
    NullInjectorError: No provider for MatDialog!
    via chaine: DataSharingService -> UtilsIhmService -> MatDialog
  Correctif :
    Soit ajouter MatDialogModule + BrowserAnimationsModule dans imports,
    soit stubber UtilsIhmService : { provide: UtilsIhmService, useValue: {} }
